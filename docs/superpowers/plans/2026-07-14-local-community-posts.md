# 로컬 커뮤니티 글 구현 계획

1. 로컬 글 저장 계약과 실패 테스트를 구현한다.
2. 저장 글을 안전한 피드 항목으로 변환하는 모델을 추가한다.
3. 홈 작성창을 저장·복원·동기화·삭제 흐름에 연결한다.
4. 로컬 글 전용 상세 상태와 반응·댓글·삭제 흐름을 구현한다.
5. 개인정보 설명과 전체 로컬 데이터 삭제에 새 키를 포함한다.
6. 모바일 E2E에서 작성, 상세 이동, 새로고침 복원, 삭제를 검증한다.
7. 전체 Vitest, E2E, lint, build, audit 후 main에 푸시한다.
8. GitHub CI와 Vercel 배포를 확인한다.
