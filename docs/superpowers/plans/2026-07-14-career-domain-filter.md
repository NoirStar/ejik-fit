# 내 커리어 희망 기술 분야 필터 구현 계획

1. 실제 스킬 그래프 응답에서 분야 선택지를 검증·정규화하는 모델 테스트를 추가한다.
2. 커리어 페이지가 기술 통계와 그래프를 독립적으로 불러오고 부분 실패를 보존하도록 테스트한다.
3. 비교 payload와 결과 범위 모델에 선택 분야를 테스트 주도로 연결한다.
4. 커리어 화면에 접근 가능한 분야 select와 부분 실패 안내를 추가한다.
5. 모바일·태블릿 브라우저 테스트에서 요청 payload, 터치 크기, 가로 넘침을 검증한다.
6. 전체 백엔드·웹·E2E·프로덕션 빌드를 실행하고 리뷰 후 `main`에 병합·푸시한다.
