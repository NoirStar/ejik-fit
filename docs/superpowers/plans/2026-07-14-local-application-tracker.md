# 로컬 지원 관리 구현 계획

1. `job-application-stages` 저장 모듈을 실패 테스트부터 구현한다.
2. 저장 보관함 테스트에 지원 단계 선택, 지원 탭, 저장 해제 정리를 추가한다.
3. 공고 카드에 로컬 지원 단계 컨트롤을 추가하고 네 번째 탭을 연결한다.
4. 모바일 탭과 단계 컨트롤 스타일을 44px 기준으로 조정한다.
5. E2E에서 단계 저장·새로고침·지원 탭·390/820/1440px를 확인한다.
6. 전체 Vitest, E2E, lint, build, audit를 실행한다.
7. 코드 리뷰 후 `main`에 병합·푸시하고 운영 배포를 확인한다.
