# 로컬 글 저장 보관함 연동 구현 계획

1. 저장 커뮤니티 모델 테스트에 local/mock/누락 ID 분류를 먼저 추가한다.
2. 저장 커뮤니티 모델이 두 출처를 보존해 화면 모델로 변환하도록 구현한다.
3. 저장 보관함 컴포넌트 테스트에 로컬 글 복원·동기화·저장 해제 실패를 추가한다.
4. 로컬 글 저장소 구독과 출처별 카드·안내 문구를 구현한다.
5. 모바일 작성→저장→보관함→상세→저장 해제 E2E를 추가한다.
6. 전체 단위 테스트, 타입, lint, audit, build, E2E와 시각 검토를 실행한다.
7. 검증된 커밋만 `main`에 fast-forward 병합·푸시하고 CI/Vercel/운영을 확인한다.
